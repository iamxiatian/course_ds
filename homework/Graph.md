# 数据结构课程作业（图算法综合）

## 作业主题
给定一个无向带权图，完成图的表示、遍历、最小生成树（Prim算法）和单源最短路径（Dijkstra算法）的编程实现，并进行可视化展示。

## 核心作业要求

### 一、图的基本信息与手工记录（文档部分）

#### (1) 画出该图（3分）
- 顶点集合：\( V = \{1, 2, 3, 4, 5, 6\} \)
- 边集合（无向带权,前2个数字代表顶点，第3个代表权值）：

```plaintext
 (1,2,5),
 (1,3,2),
 (1,4,3),
 (2,3,1),
 (2,5,6),
 (3,4,4),
 (3,5,7),
 (4,5,2),
 (4,6,8),
 (5,6,3)
```

- 要求：使用draw.io或其他任意绘图工具绘制该图，标出所有顶点及边上的权值，并导出为`graph.png`。

#### (2) 写出该图的邻接矩阵和邻接表表示（4分）
- **邻接矩阵**：按顶点编号 1~6 顺序给出 \(6 \times 6\) 矩阵，无直接相连的顶点用无穷大（或 0）表示，请明确说明表示方式（例如用 0 或 ∞ 表示无边）。
- **邻接表**：每个顶点的邻接点按顶点编号升序排列，格式如：
  ```
  1: 2(5), 3(2), 4(3)
  2: 1(5), 3(1), 5(6)
  ...
  ```

### 二、编程实现要求

#### 1. 图的遍历（4分）
- 从顶点 **1** 出发，按照顶点编号升序的顺序访问邻接点。
- 分别实现：
  - 深度优先遍历（DFS）：输出顶点访问序列
  - 广度优先遍历（BFS）：输出顶点访问序列
- 要求：代码独立实现图的存储（邻接矩阵或邻接表），遍历结果与手工推导一致。

#### 2. Prim算法构造最小生成树（5分）
- 从顶点 **1** 开始构造该图的最小生成树（MST）。
- 要求输出 **每一步** 加入的边及顶点集合 \(U\) 的变化过程（文字或表格形式）。
- 画出最终的MST（可手绘后拍照，或使用工具生成，保存为 `mst.png`）。

#### 3. Dijkstra算法求最短路径（4分）
- 求从顶点 **1** 到其余各顶点的最短路径长度和具体路径。
- 输出格式示例：
  ```
  1 -> 2: 最短长度 = 3，路径 = 1→3→2
  1 -> 3: 最短长度 = 2，路径 = 1→3
  ...
  ```

#### 4. 可视化（加分/核心要求）（计入总分）
- 要求实现 **可视化界面**，展示：
  - 原始图（顶点、边、权值）
  - 可选择演示DFS、BFS 遍历过程（高亮访问顺序）
  - 可选择演示Prim 算法构造 MST 的每一步
  - 可选择演示Dijkstra 算法的最短路径结果
- 可采用 **Vibe Coding** 方式（即借助大模型生成可视化代码，可以使用Claude Code工具），但需遵循下方“AI辅助开发要求”。
