# 数据结构课程作业（迷宫求解）

## 作业主题
使用栈、队列和递归三种方式实现迷宫求解问题。

## 核心作业要求
### 一、编程实现要求
1. 迷宫表示
   - 使用二维数组（或自定义结构）表示迷宫，其中：  
     `0` 表示通路，`1` 表示墙壁，`2` 表示已走过的路径。  
   - 迷宫大小至少为10×10，且包含至少一条从起点到终点的可行路径。  
   - 起点默认为`(1,1)`，终点默认为`(n-2, m-2)`（即迷宫边界内层）。

2. 三种求解方式 
   必须分别独立实现以下三种方法，并输出各自的路径结果（坐标序列）及路径长度（步数）：  
   - 栈实现（DFS）：利用栈的后进先出特性，模拟深度优先搜索。  
   - 队列实现（BFS）：利用队列的先进先出特性，模拟广度优先搜索（保证找到最短路径）。  
   - 递归实现（DFS）：通过函数递归调用进行探索与回溯。

3. 数据结构  
   - 栈和队列可以自己实现，也可以使用语言本身提供的相关结构或者第三方实现，但需要在文档中说明。

4. 代码规范  
   - 完整注释（函数功能、参数说明、关键逻辑，采用docstring方式）。  
   - 异常处理：能处理迷宫无解的情况（输出“无可行路径”）。  
   - 输出格式清晰，例如：  
     ```
     栈求解路径：(1,1)->(1,2)->...->终点，共X步  
     队列求解路径：(1,1)->...->终点，共Y步（最短）  
     递归求解路径：...  
     ```

3. 禁止直接复制AI生成的完整代码，不经验证就提交作业，一经发现按作业无效处理。
